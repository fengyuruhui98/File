

# 1.原地排序奇偶
有一数组:  
a[8] = {56,68,20,15,17,101,132,119,}  
对该数组进行排序使得奇数全部在偶数的前面，并且不可以引进分配新的数组空间。  

```c
void sort_four(int * a, int len){
    int i = 0, j, temp;
    j = len - 1;
    while(i < j)if(a[i] % 2 == 0) i++;else if(a[j] % 2 == 1) j--; else{
        temp = a[i];
        a[i] = a[j];
        a[j] = temp;
    }
}
```
